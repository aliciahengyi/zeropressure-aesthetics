# Zero-Pressure Aesthetic · 零压审美

A calm, minimal, Morandi-toned visual philosophy — distilled from building a "zero-pressure" planning app.

> **Core belief:** put people at ease, not on edge. Every color, line, and element serves quiet — not attention-grabbing, not urgent, not judging. When in doubt, remove it.

**In one breath:** one hue family (Morandi), depth over variety. No boxes or borders by default — only buttons may have one, and cards are a fallback rather than the default. One accent color for emphasis, used sparingly. Small, restrained controls. Generous whitespace. Copy that is warm and non-commanding — companionship, not instruction.

Medium-agnostic: works for UI (app / web / mini-program), posters, slides, documents, and charts alike.

## What's inside

- **Color** as semantic tokens (`bg` / `ink` / `accent` / `line` / layers) + contrast guidance, with **six ready-made Morandi palettes** (each named after its two dominant hues)
- **No boxes** by default, and a strict priority order for dividing sections when you need to: hairline > side color strip > accent-color heading > box (box is the last resort, thin rounded, unfilled)
- **Accent color for emphasis** — never borders or big fills
- **Small, restrained controls** — a thin-line switch component with copy-paste CSS, plus a rule: once a control's style is changed and confirmed, it becomes the global default — never re-litigated per instance
- **Typography** hierarchy and **whitespace** rhythm
- **Voice & wording** — gentle, a little poetic, yet instantly clear; replace command words with warm neutral ones
- **Do's & Don'ts**
